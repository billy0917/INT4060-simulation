Learning Objectives

Exploring Area

Learn how to measure the space inside flat shapes

What is Area?

Area is the amount of space inside a flat (2D) shape. We measure area in square units — think of it as counting how many small squares fit inside the shape.

Rectangle: Area = width × height Square: Area = side × side

For example, a rectangle with width 5 and height 3 has an area of 5 × 3 = 15 square units.

Try the Simulator

Drag the sliders to change the shape. Watch the colourful unit squares fill up and count them.

Guided Activity

Follow these steps in the simulator above:

  1. Set the shape to Rectangle. Set width = 5 and height = 3. How many unit squares do you see? (Answer: 15)
  2. Now change the width to 6. What happens to the area? Does it get bigger or smaller?
  3. Switch to Square. Set the side length to 4. What is the area? (Answer: 16)
  4. Try the Challenge mode — adjust the sliders to match the target area.

Think About It

  • If you double the width of a rectangle, what happens to the area?
  • Can two different rectangles have the same area? Try to find an example.

Exploring Volume

Learn how to measure the space inside 3D shapes

What is Volume?

Volume is the amount of space inside a 3D shape. We measure volume in cubic units — imagine stacking small cubes to fill the shape.

Cuboid: Volume = length × width × height Cube: Volume = side × side × side

For example, a cuboid with length 4, width 3, and height 2 has a volume of 4 × 3 × 2 = 24 unit cubes.

Try the Simulator

Drag the sliders to change the shape. Rotate the 3D model by dragging on it. Try "Build Layer by Layer" to watch the cubes stack up.

Guided Activity

Follow these steps in the simulator above:

  1. Set the shape to Cuboid. Set length = 4, width = 3, height = 2. Count the unit cubes. (Answer: 24)
  2. Click Build Layer by Layer to watch the cubes stack up. How many cubes are in each layer? (Answer: 12)
  3. Switch to Cube. Set the side length to 3. What is the volume? (Answer: 27)
  4. Drag to rotate the 3D shape and view it from different angles.
  5. Try the Challenge mode — adjust the sliders to match the target volume.

Think About It

  • How is volume different from area?
  • If you double the height of a cuboid, what happens to the volume?
  • A cube has side length 2. How many unit cubes fit inside? (Answer: 8)

What We Learned

A recap of the key concepts and formulas

Area

Rectangle: Area = width × height
Square: Area = side × side
  • Area measures the flat space inside a 2D shape
  • Measured in square units
  • Count the unit squares to find the area

Volume

Cuboid: Volume = length × width × height
Cube: Volume = side × side × side
  • Volume measures the space inside a 3D shape
  • Measured in cubic units
  • Count the unit cubes to find the volume

Key Takeaways

  • Area tells us how much flat space a 2D shape covers
  • Volume tells us how much space a 3D shape fills
  • You can always count unit squares (for area) or unit cubes (for volume) to check your answer
  • When one dimension doubles, the area or volume also doubles
